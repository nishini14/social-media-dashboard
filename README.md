# 📊 Social Media Advertisement Performance Dashboard

**Developed by: Nishini R**

---

## 📌 Project Overview

This project provides a comprehensive analysis of social media advertisement performance using Power BI. By integrating datasets related to campaigns, users, and ad events, the dashboard delivers clear insights into key performance metrics such as impressions, clicks, engagements, and purchases.

---

## ❓ Problem Statement

Analyzing ad campaign performance manually is difficult due to large volumes of data. This dashboard provides an interactive interface to:

* Track impressions, clicks, and engagement metrics
* Compare performance across platforms (Facebook & Instagram)
* Analyze user behavior based on age and gender
* Evaluate campaign effectiveness and conversion rates

---

## 🛠️ Tools Used

| Tool               | Purpose                                    |
| ------------------ | ------------------------------------------ |
| Power BI           | Data Modeling, DAX, and Dashboard Creation |
| Power Query Editor | Data Cleaning and Transformation           |
| Excel              | Initial Data Preparation                   |

---

## 📂 Dataset Information

The project uses multiple datasets:

* **ad_events.csv** – Impressions, clicks, engagements, purchases
* **ads.csv** – Advertisement and campaign mapping
* **campaigns.csv** – Campaign-level data
* **users.csv** – User demographics
* **social_media.csv** – Platform details
